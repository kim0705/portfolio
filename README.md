## 💡 Portfolio Structure

### 🏠 Home
포트폴리오의 첫 화면으로 개발자 소개와 프로젝트 미리보기를 제공합니다.

- **Hero Section**
  - 개발자 소개 및 핵심 메시지 표시

- **Floating Profile Side Tab**
  - 프로필 이미지
  - 연락처 및 GitHub 링크
  - 주요 기술 스택 태그

- **Project Preview Cards**
  - 프로젝트 썸네일과 요약 정보를 카드 형태로 제공
  - 카드 클릭 시 Project Detail Page로 이동

---

### 👤 About
개발자에 대한 정보를 카드 레이아웃으로 구조화했습니다.

- **Perspective**
  - 개발에 대한 가치관과 문제 해결 방식

- **Skills**
  - 사용 가능한 기술 스택 정리

- **Career**
  - 참여 프로젝트 및 업무 경험

- **Education**
  - 학력 정보

- **Certification**
  - 보유 자격증

---

### 📂 Projects Page
프로젝트 목록을 카드 형태로 구성하여  
각 프로젝트의 핵심 내용을 한눈에 확인할 수 있도록 구현했습니다.

각 카드에는 다음 정보가 포함됩니다.

- 프로젝트 이름
- 사용 기술 스택
- 프로젝트 요약
- 썸네일 이미지

카드를 클릭하면 **Project Detail Page**로 이동합니다.

---

### 📄 Project Detail Page

각 프로젝트의 상세 내용을 확인할 수 있는 페이지입니다.  
프로젝트를 단순 소개가 아닌 **문제 해결 과정 중심으로 정리했습니다.**

다음 내용을 포함합니다.

- 프로젝트 개요
- 기술 스택
- 주요 담당 역할
- **Flow Chart 기반 핵심 로직 설명**
- **서비스 시연 영상 (Demo, 제공 가능한 경우)**
- **AS-IS → TO-BE 개선 과정**
- 트러블슈팅 사례

또한 프로젝트의 핵심 로직을 **Flow Chart로 시각화하여  
데이터 흐름과 처리 과정을 이해하기 쉽게 정리했습니다.**
